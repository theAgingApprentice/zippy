Overview

This repository contains Arduino code that will get you off to a flying start when you begin your next robot project. Simply clone this template repository and use the source code to boot strap the development of your robot's integrated circuit firmware.

Features

  1. Modular code for easy maintenance.
  2. Hardware aware for easy configuration.
  3. 802.11 wifi capable.
  4. OTA software update capable.
  5. Web interface.
  6. MQTT QOS 1 support.
  7. Repo auto generates Doxygen online documentation.
